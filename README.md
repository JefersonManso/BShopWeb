# BShop Monorepo

Este repositório contém dois projetos:

1. **BShop.ProductApi** - Microsserviço ASP.NET Core Web API para gerenciamento de produtos e categorias.
2. **BShop.Web** - Aplicação ASP.NET Core MVC que consome a ProductApi.
